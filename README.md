<h2 align="center">FANYI</h2>

FANYI是一款命令行翻译工具

能将中文自动翻译成英文，英文自动翻译成中文

支持驼峰、下划线、中线命名方式

------

## 使用

![demo](./doc/demo.gif)

## 安装

```bash
git clone https://github.com/tanghuibo/fanyi.git
cd fanyi
npm install gfanyi -g
```
