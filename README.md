# FANYI

命令行翻译工具， 中文自动翻译成英文，英文自动翻译成中文

英文支持驼峰、下划线、中线命名方式

------

### 使用
输入命令: fanyi

然后输入你要翻译的内容

```bash
>>> fanyi           
power by youdao
> helloWorld
---------------------------------
hello World
你好，世界
---------------------------------
> 你好，世界
---------------------------------
你好，世界
Hello, world
---------------------------------
```

### 安装

```bash
git clone https://github.com/tanghuibo/fanyi.git
cd fanyi
npm install . -g
```
